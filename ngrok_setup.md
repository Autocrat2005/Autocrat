# Ngrok Setup for Autocrat CLI

Ngrok is used to expose the local Autocrat web server to the internet, enabling remote control capabilities from devices such as your phone.

## Steps to Set Up Ngrok

1. **Download and Install Ngrok**:
   - Visit [ngrok.com](https://ngrok.com) and download the appropriate version for your operating system.
   - Extract the downloaded file and add it to your system's PATH for easy access.

2. **Start the Autocrat Web Server**:
   - Run the following command to start the web server:
     ```bash
     python main.py --web
     ```

3. **Expose the Web Server Using Ngrok**:
   - Open a new terminal and run:
     ```bash
     ngrok http 9000
     ```
   - This will generate a public URL (e.g., `https://<random>.ngrok.io`) that maps to your local server.

4. **Access Autocrat Remotely**:
   - Use the public URL generated by ngrok to access the Autocrat interface from your phone or any other device.

## Notes

- Ensure that the Autocrat web server is running before starting ngrok.
- Keep the ngrok terminal open while using the public URL.
- For added security, consider using ngrok's authentication features.
